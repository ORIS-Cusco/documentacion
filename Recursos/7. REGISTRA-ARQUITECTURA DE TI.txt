I. INTRODUCCIÓN
El presente documento ayuda a describir las necesidades de recursos de hardware y software del Sistema de Registro Académico “Registra”.

1.1. Propósito
El objetivo es brindar información necesaria sobre los recursos que se requiere para el Sistema de Registro Académico.

1.2. Alcance
Descripción de recursos de hardware y software del proyecto Sistema de Registro Académico. 

1.3. Definiciones y abreviaturas
Se definen los siguientes términos para mejor interpretación del contenido de este documento.

1.3.1. Definiciones
- MINEDU: Ministerio de educación.
- Sistema de Gestión Docente: Es el sistema para el registro de información de los procesos que intervienen los docentes contratados y nombrados de las instituciones de educación superior tecnológica.

1.3.2. Abreviaturas
    MINEDU: Ministerio de Educación.
    DRE: Dirección Regional de Educación.
    UGEL: Unidad de Gestión Educativa Local.
    UE: Unidad Ejecutora.
    HTTP: Hypertext Transfer Protocol - Protocolo de comunicaciones.
    LAN: Red de área local.
    WAN: Red de área amplia
    VPN: Red Privada Virtual
    IIS: Internet Information Services.
    MS SQL Server: Motor de Base de datos relacional.
    GRANJA: Conjunto de Servidores distribuidos.
    DMZ: Zona desmilitarizada.
WEB API: Es una extensión del framework.net para construir servicios basados en HTTP que pueden ser accedidos por diferentes aplicaciones en distintas plataformas tales como entornos web, Windows y móviles.
Web Farm Framework: Microsoft Web Farm Framework es un producto gratuito, completamente soportado, de Microsoft que permite administrar una granja de servidores web, permite automatizar la instalación y configuración de componentes de la granja, y permite sincronizar automáticamente y desplegar aplicaciones ASP.NET.
ONLine: Aplicaciones de negocio cuyo núcleo se encuentra expuesto en internet.

1.4. Referencias
Para la elaboración del presente documento se ha tomado como referencia estándares de OTIC, cuyos documentos son:
* Guías para el desarrollo de aplicaciones distribuidas basadas en Microsoft .Net
* Estándares para el uso de herramientas de desarrollo y plataformas de Aplicaciones WEB .Net (OTIC).
* Microsoft Web Farm Framework: 
    http://www.iis.net/downloads/microsoft/web-farm-framework
